# GW-Market

Marketplace for players of Guild Wars 1

This is a public local version of the site, with fake shops, requiring no database.

# Get Started

The repo has a client and a server side. it is advised to open a terminal on each side.

## Client/Front

-   cd ./client
-   npm i
-   npm run watch

## Server/Back

-   cd ./server
-   npm i
-   npm run start

## Use the website

Simply open your browser with localhost and default port 3026

-   localhost:3026
